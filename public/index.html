Interactive demonstration of all available API endpoints

Get All Questions
GET
/api/questions
Retrieves all questions from the database (without answers). Each question includes the timeLimit field (60 seconds by default).
Get Random Question
GET
/api/question/random
Gets a random question with optional filtering by categories and exclusion of recent questions. Includes 1-minute time limit for answering.
Get Categories
GET
/api/questions/categories
Returns all available question categories from the database.
Random Question by Category
GET
/api/question/random?categories=...
Gets a random question from a randomly selected category. Demonstrates the category filtering functionality of the random question endpoint.
Check Answer
POST
/api/answer
Validates a submitted answer for a specific question. Requires questionId and selectedOption in the request body.
